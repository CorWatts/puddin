# Puddin

Everyone loves pudding.

This was a poem posted on [Reddit](https://www.reddit.com/r/ShittyPoetry/comments/3t2qje/how_do_you_describe_pudding/) written by [frankieh](https://www.reddit.com/user/frankieh)
